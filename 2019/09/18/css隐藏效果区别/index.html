css隐藏效果区别

Just a blog site
2019-09-18
css

使用 css 实现隐藏效果主要是用到一下方法：

  1. display: none
  2. visibility: hidden
  3. opacity: 0

虽然三者都能实现元素的隐藏，但是存在着些许的区别

隐藏方式 是否占位 绑定到隐藏元素上的事件是否生效
display: none 否 否
visibility: hidden 是 否
opacity: 0 是 是
